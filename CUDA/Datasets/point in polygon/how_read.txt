A point in polygon file must contain:
1. title: Latitude, Longitude
2. Total number of points N
3. The query point (only one point)
4. Coordinates of the spherical polygon (size: N)

Example:
Latitude,Longitude
4

-12.071903,-77.126787
-12.078513,-77.104394
-12.078973,-77.093252
